# Student Management System (C++)

## Overview
A console-based Student Management System developed in C++. This project allows users to manage student records using Object-Oriented Programming and file handling.

## Features
- Add Student
- View Students
- Search Student
- Update Student
- Delete Student
- Save Data to File
- Load Data from File

## Technologies Used
- C++
- Object-Oriented Programming (OOP)
- File Handling
- STL (Vector)
